Shoppistae

Shoppistae is a modern e-commerce web application built using React, designed to provide a seamless and enjoyable shopping experience. It features a user-friendly interface, smooth navigation, secure authentication, and a robust cart and checkout system. With real-time stock management through an admin panel, the platform ensures efficiency for both users and sellers. Stock Sync

StockSync is a stock market application built using Flask, a Python-based web framework, designed to help users manage their stock portfolio and make informed decisions. The application offers functionalities like user registration, login, adding stocks to a wishlist, predicting stock prices using machine learning, and fetching real-time stock data. It utilizes MongoDB for efficient data storage, yfinance for retrieving live stock information, and scikit-learn for implementing linear regression models for price prediction. This project aims to provide users with insightful stock predictions and a seamless management experience.
